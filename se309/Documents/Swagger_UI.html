user

GET/users
Get a list of all user objects stored in the database.
PUT/users/{id}/addBalance/{b}
Add a float value to a user's balance.
GET/users/{id}/getBalance
Get balance of the user with the given ID.
GET/users/{id}/getPassword
Get password of the user with the given ID.
GET/users/{id}/getUsername
Get the username from the user with the given ID.
PUT/users/{id}/setUsername/{u}
Set the username of a user by providing the ID.
PUT/users/{id}/subBalance/{b}
Subtract a float value from a user's balance.
DELETE/users/delete/{id}
Delete the user with a certain ID.
GET/users/login/{u}/{p}
Verify that the username and password passed in are equivalent to a user's in the database.
POST/users/new/{u}/{p}
Create a new user with the username and password values passed in.
